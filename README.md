🧶 Tomomi's To-Do 🧶

Exam/Culminating season or just a lot of work on your plate? Use this cute to-do list website (featuring Tomomi the grey cat!!) to write down your tasks and make things less overwhelming 💗

------
Assignments have been piling up for me since it's about to be exam/culminating season 🙁 Apple's reminders app now brings me a lot of trauma because I've been using it for a long time and seeing a million tasks build up there is just overwhelming... so I made this cute to-do website in hopes that it'll inspire me to study, not procrastinate, and actually do work 🥹

------
I made this website using HTML, CSS, and JS. I feel like I'm getting pretty familiar with these languages (except javascript of course, I still need to work on that 😞). Also, I think I really like modals because I used it like twice here, but it's honestly so fun to see text pop up when you click something! This time, I wasn't indecisive about the styles and I stuck to Tomomi's color theme/style the whole time I worked on this project :) I used quite a bit of animations in this project as well.

------
Javascript was the main challenge. I'm still learning how to use it. I also really want to know how to organize/structure my javascript file because I just feel like it's messy right now. I learned that the order of functions doesn't matter because they will run first anyways, but I don't know about the other stuff. For example, if I put the code for a modal under the other, which one runs first? I think when I'm familiar with this, I'll be able to use javascript more efficiently.
